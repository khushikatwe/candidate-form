
# Candidate Registration Form

Simple Candidate Registration Form built with React (via CDN) â€” runs without build tools.

## Features
- Add candidate (Name, Email, Phone, Position, Experience, Status, Resume link)
- Stores candidates in localStorage
- Remove individual candidate / Clear all
- Clean, responsive UI

## How to use
1. Open the site (GitHub Pages URL) or open `index.html` locally in your browser.
2. Fill the form and click **Add Candidate**. Saved candidates appear below.

## Tech
- React + ReactDOM (via CDN)
- Babel (in-browser JSX)
- Plain CSS

## Note
This is a demo prototype for HR workflow. For production, move to a proper React build and server for persistent storage.
